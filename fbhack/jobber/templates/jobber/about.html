About our project

Problem which we solved

Problem about automation future jobs taken by AI.

We created web service which checks if your current job is at risk of automation for next 20 years.

If the answer is yes then user goes through personality analysis based on text stream big5.

Then our service propose you future and safe job. Also our app provides you with study path in order to get this job (courses, articles, videos)

Technology stack

- Machinelearning analysis of big-5 personality testes trained on myPersonality/Cambridge Analitica datasets , we created our own model for that.

- Simulating automation of jobs for next 20 years with salaries

- Django framework for Backend and Frontend

- Postgresql database

- Youtube api

- Webscraping on Quora

- Googleappengine for deployment our app
